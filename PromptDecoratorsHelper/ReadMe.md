# 🚀 Prompt Decorators Helper (PDH) v1.0

**Prompt Decorators Helper**는 ChatGPT, Claude, Perplexity, Grok 등 다양한 AI 채팅 서비스에서 자주 사용하는 프롬프트 명령어(데코레이터)를 **클릭 한 번으로 입력**해주는 크롬 확장 프로그램입니다.

최신 AI 서비스들의 복잡한 에디터 환경(React, ProseMirror 등)에서도 텍스트가 누락되거나 개행이 무시되지 않도록 **완벽한 호환성**을 제공합니다.

## ✨ 주요 기능

  * **🎯 초소형 플로팅 아이콘 (22px)**
      * 화면을 가리지 않는 아주 작은 크기입니다.
      * **드래그 & 도킹**: 아이콘을 드래그하면 가장 가까운 화면 모서리(좌/우/상/하)에 자석처럼 붙습니다.
      * **위치 기억**: 사이트마다 아이콘을 놓아둔 위치를 기억합니다.
  * **🧠 스마트 모달 시스템**
      * 아이콘이 화면 왼쪽에 있으면 모달은 오른쪽에, 아이콘이 오른쪽에 있으면 모달은 왼쪽에 뜹니다. (화면 밖으로 잘림 방지)
  * **💬 툴팁 설명**
      * 명령어 위에 마우스를 올리면 상세 설명이 **검은색 말풍선**으로 깔끔하게 뜹니다. (목록이 길어지지 않음)
  * **⚡ 하이브리드 입력 방식**
      * **데코레이터 추가**: 실제 사람이 타이핑하는 것처럼 입력하여 Perplexity 등에서도 **개행과 입력이 완벽하게 동작**합니다.
      * **초기화**: 긴 텍스트는 붙여넣기 방식으로 순식간에 입력됩니다.
  * **🛡️ Shadow DOM 적용**: 웹사이트의 스타일이 깨지거나, 웹사이트의 스타일이 확장 프로그램에 영향을 주지 않습니다.

## 🛠️ 설치 방법

1.  **폴더 생성**: 컴퓨터에 폴더를 하나 만들고(예: `PDH_Extension`), 위 3개 파일을 저장합니다.
2.  **크롬 확장 프로그램 관리 접속**:
      * 주소창에 `chrome://extensions/` 입력 후 엔터.
3.  **개발자 모드 켜기**:
      * 우측 상단의 **"개발자 모드"** 스위치를 켭니다.
4.  **압축해제된 확장 프로그램을 로드합니다**:
      * 좌측 상단 **"압축해제된 확장 프로그램을 로드합니다"** 버튼 클릭.
      * 1번에서 만든 폴더를 선택합니다.
5.  **설치 완료 & 새로고침**:
      * 확장 프로그램 목록에 `Prompt Decorators Helper`가 추가되었는지 확인합니다.
      * **반드시** 사용하려는 AI 사이트(ChatGPT, Perplexity 등)를 **새로고침(F5)** 해야 적용됩니다.

## 📖 사용 가이드

### 1\. 아이콘 이동 및 도킹

  * 화면에 떠 있는 초록색 아이콘을 드래그하여 원하는 위치로 옮기세요.
  * 손을 떼면 가장 가까운 벽(Edge)으로 스르륵 이동하여 붙습니다.

### 2\. 데코레이터 입력

  * 아이콘을 클릭하면 메뉴창이 열립니다.
  * 원하는 명령어(예: `++요약`)를 클릭하면 입력창에 자동으로 타이핑됩니다.
  * 마우스를 명령어 위에 올리면 해당 기능에 대한 설명이 **툴팁**으로 뜹니다.

### 3\. 아이콘 숨기기 및 복구 (중요\!)

  * **숨기기**: 아이콘에 마우스를 올리면 나타나는 작은 **`×`** 버튼을 누르면 아이콘이 사라집니다.
  * **복구하기**: 아이콘이 사라졌다면, 브라우저 우측 상단의 **퍼즐 모양(🧩) → Prompt Decorators Helper 아이콘**을 클릭하세요.
      * *"아이콘이 활성화되었습니다"* 라는 메시지와 함께 다시 나타납니다.

### 4\. 초기화 기능

  * 메뉴 최상단의 **"초기화 (전체 입력)"** 버튼을 누르면, GitHub에 있는 원본 파일 전체 내용을 입력창에 붙여넣습니다. (프롬프트 엔지니어링 템플릿 전체 로드용)

## ❓ 문제 해결 (Troubleshooting)

**Q. 아이콘이 안 보여요\!**

> A1. 브라우저 우측 상단 확장 프로그램 목록에서 아이콘을 한 번 클릭해보세요.
> A2. 그래도 안 보이면 `chrome://extensions`에서 확장 프로그램을 "새로고침" 하고, 웹페이지도 "새로고침(F5)" 하세요.
